# 👹 DOOM OFFLINE: HELL ON WEB

Добро пожаловать в современный клон легендарного DOOM, работающий прямо в вашем браузере! Это полностью офлайн-версия игры с продвинутой графикой, разнообразным оружием и эпической битвой с боссом.

## 🚀 Основные особенности

*   **20 Уровней Ада**: Пройдите через процедурно генерируемые лабиринты и вручную созданные арены.
*   **Продвинутая 3D Графика**: Реализовано на Three.js с динамическим освещением, эффектами тумана и процедурными анимациями врагов.
*   **Продвинутый Арсенал**: 
    *   🪚 **Бензопила**: Бесконечный ближний бой для экономии патронов.
    *   🔫 **Пистолет**: Стандартное оружие (Пули).
    *   🧨 **Дробовик**: Мощный на дистанции (Дробь).
    *   💥 **Двустволка (SSG)**: Двойной ствол, двойная смерть (Дробь).
    *   🚀 **Ракетница**: Урон по площади (Ракеты).
    *   🔷 **Плазменная винтовка**: Скорострельный энергетический огонь (Ячейки).
    *   🟢 **BFG-9000**: Легендарное супероружие (Ячейки x40).
*   **Реалистичные Механики**:
    *   **Спринт**: Удерживайте Shift, чтобы бежать быстрее.
    *   **Разброс и Отдача**: Динамическая точность и физическая отдача камеры при стрельбе.
    *   **Система патронов**: Раздельные пулы боеприпасов для разного оружия.
*   **Визуальные эффекты**: Система частиц крови, тряска экрана, динамические вспышки и туман.
*   **Финальный Босс**: Схватка с Кибер-Демоном на 20-м уровне.

## 🎮 Как играть

### На ПК:
*   **WASD**: Перемещение
*   **Мышь**: Осмотр / Стрельба
*   **Пробел**: Прыжок
*   **ESC / TAB**: Пауза и настройки чувствительности

### На мобильных устройствах:
*   **Левый джойстик**: Перемещение
*   **Правая часть экрана**: Осмотр камеры
*   **Кнопки FIRE / JUMP**: Стрельба и прыжок

## 🛠 Технологии
*   [Three.js](https://threejs.org/) — 3D движок.
*   **Web Audio API** — Процедурная генерация звуковых эффектов (без внешних аудио-файлов!).
*   **Vanilla JS & HTML5 Canvas**.
